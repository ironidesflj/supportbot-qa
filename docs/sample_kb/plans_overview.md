# CloudStream Pro Plans Overview

## Available Plans

### Basic Plan
- Up to **1 simultaneous stream**
- HD (720p) video quality
- Standard library access
- Price: $7.99/month

### Standard Plan
- Up to **2 simultaneous streams**
- Full HD (1080p) video quality
- Expanded library access
- Price: $12.99/month

### Premium Plan
- Up to **4 simultaneous streams**
- 4K Ultra HD + HDR video quality
- Full library access including premium exclusives
- Offline downloads on up to 6 devices
- Price: $19.99/month

### Family Plan
- Up to **6 simultaneous streams**
- 4K Ultra HD video quality
- Full library access
- Up to 6 user profiles with parental controls
- Price: $24.99/month

## Device Compatibility

CloudStream Pro is available on web browsers, iOS, Android, Apple TV,
Roku, Amazon Fire TV, Android TV, and select smart TVs. The number of
devices that can stream simultaneously depends on your plan tier as
described above.
